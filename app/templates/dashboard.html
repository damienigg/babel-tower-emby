How to trigger a job

Three ways to start a job (besides the per-item button on the Library page):

  1. From an Emby item id (defaults from Settings; override per call as query params): 
    curl -X POST 'http://<host>:8765/api/process/<EMBY_ITEM_ID>?target_lang=fr&mode=scene'
  2. From a media path (no Emby needed): 
    curl -X POST http://<host>:8765/transcribe-translate \
  -H 'Content-Type: application/json' \
  -d '{"media_path": "/mnt/media/movie.mkv", "target_lang": "fr"}'
  3. Auto-trigger on new items: in Emby admin → Server Settings → Notifications → Add Notification → Webhook, set the URL to http://<host>:8765/webhook/emby. If you set a webhook_secret in Settings, add the matching X-Babel-Token header in Emby's webhook config.
