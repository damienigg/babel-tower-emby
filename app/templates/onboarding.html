Connect to your media server

Subtitle This reads your library from Emby, Jellyfin, or Plex. Pick the one you run, paste its URL and an API key, then click Test connection below to verify it works before saving.

The address your media server's web UI runs on. For Plex this is typically port 32400; Emby and Jellyfin default to 8096.

Emby/Jellyfin: Dashboard → API Keys → New API Key. Plex: grab your X-Plex-Token from any browser request to your server (see Plex's docs). {% if values.get('media_server_api_key') == '[set]' %} (An API key is already saved — leave this blank to keep it.) {% endif %}

Pick your defaults

Which language do you want subtitles in, how much should the app do per cue, and which translator should it use. You can override any of these per-job from the Library, so just set the most common combination here.

audio is the recommended starting point. Scene and cinematic require an LLM-based provider (see below) and a vision-capable model.
